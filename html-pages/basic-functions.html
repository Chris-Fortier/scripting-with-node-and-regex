JavaScript Functions: Basic

functions documented

add - Add two values together

subtract - subtract a number from another number

multiply - multiply a number by another number

divide - divide a number by another number

increment - increments a number by 1

decrement - decrements a number by 1

remainder - get the remainder of a number divided by another number

multiplyDecimal - multiply a decimal number by another number

divideDecimal - divide a decimal number by another number

concatenate - concatenates input1 and input2 using + operator

concatenatePlusEquals - concatenates input1 and input2 using += operator

madLib - concatenates the three strings using variables

getLengthOfString - counts how many letters are in your string

getFirstLetterOfString - returns the first letter of your string

getNthLetterOfString - returns the nth letter of your string

getLastLetterOfString - returns the last letter of your string

getNthToLastLetterOfString - returns the nth to last letter of your string

push - pushes a newItem to the end of an array

pop - removes the last element of an array and returns what is left

shift - removes the first element of an array and returns what is left

unshift - adds the newItem to the beginning of the array

addToShoppingList - adds itemType and quantity to the shopping list

nextInLine - adds item to end of array, removes item from the beginning, returns the new array
