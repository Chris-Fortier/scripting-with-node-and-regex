JavaScript Functions: Algorithms

functions documented

convertToF - Convert Celsius to Fahrenheit

reverseString - Reverse a String

factorialize - Factorialize a Number

findLongestWordLength - Find the Length of the Longest Word in a String

largestInEachSubarray - Return Largest Numbers in Arrays

confirmEnding - This checks if string ends with target.

repeatStringNumTimes - This returns a new string that is str repeated num times.

truncateString - This will return a truncated string with ... after it if its longer than num.

findElement - This looks through an array and returns the first element that passes the function func = (num) => num % 2 === 0.

booWho - This checks if the argument is a boolean value.

titleCase - This makes each word of a given sentence start with a capital letter but makes the rest of the letters lowercase.

frankenSplice - This copies each element of the first array into the second array, in order, beginning inserting elements at index n of the second array.

bouncer - This will remove all falsy values from an array.

getIndexToIns - This will return the lowest index at which a value (second argument) should be inserted into an array (first argument) once it has been sorted.

mutation - This will return true if the string in the first element of the array contains all of the letters of the string in the second element of the array.

chunkArrayInGroups - This splits an array (first argument) into groups the length of size (second argument) and returns them as a two-dimensional array. Look at the console to verify that its a 2D array.
